◆ゲームの概要
	タイトル	ぎゃくれおに
	制作人数	6人
	制作環境	Visual Studio 2015	DirectX9.0
	制作期間	4ヶ月
	使用言語	C言語
	動作環境	Windows10
	
◆操作方法
  ◇キーボード
  
    ・プレイヤー1人目(人間)
	W.A.S.Dキー	:	移動
	
	・プレイヤー2人目(人間)
	U.H.J.Kキー	:	移動
	
	・プレイヤー3人目(オニ)
	矢印キー	:	移動
	7(テンキー)	:	変身(電柱)
	8(テンキー)	:	変身(ダンボール)
	9(テンキー)	:	変身(ゴミ箱)
	
  ◇ゲームパット(DirectInput)
  
    ・プレイヤー(人間)
    左スティック:	移動
    
    ・プレイヤー(オニ)
     左スティック:	移動
     Yボタン	:変身(ダンボール)
     Xボタン	:変身(電柱)
     Aボタン	:変身(ゴミ箱)
