◆操作方法
  ◇キーボード
    ・i匹目
      W.A.S.D	:	移動
	・2匹目
	  矢印キー	:	移動
	・共通
	  SPACE		:	射撃
	  Cキー		:	結合
	  Vキー		:	分解
	  
  ◇ゲームパット(DirectInput)
    ・1匹目
       左スティック	:	移動
       LT			:	射撃
    ・2匹目
       右スティック	:	移動
       RT			:	射撃
    ・共通
      LB			:	結合
      RB			:	分解
      ※結合時RTで射撃可能
